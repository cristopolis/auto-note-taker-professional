## Limitation

1. GPU acceleration only works on CUDA environment.

Recommended VRAM values.

| Precision | Whisper model | Required VRAM |
| :-------: | :-----------: | :-----------: | 
|     1     |    tiny     |     ~1 GB     |
|     2     |    base     |     ~1 GB     |
|     3     |    small    |     ~2 GB     |
|     4     |   medium    |     ~5 GB     |
|     5     |    large    |    ~10 GB     |
